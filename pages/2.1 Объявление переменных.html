2.1 Объявление переменных


Для начала работы вы можете использовать прошлый сохранённый файл с готовой структурой как готовый шаблон.

Можно видоизменить приветствие и добавить переменную,которую можно вывести вместе с сообщением. Данной переменной задаётся начальное значение, оно может быть изменено в коде. Но пользователь этого сделать не может.

Для объявления переменной используйте оператор "let". Объявите переменную с именем "name":

Затем присвойте переменной ваше имя и выведите сообщение с использованием вашей переменной.

Пример выполненного скрипта: 

1
2
3
4
<script type="text/javascript">
   let name = 'Ильшат';
   alert('Привет, '+ name);
</script>

Сохраните файл в личной папке под названием 2.1 Объявление переменных.
