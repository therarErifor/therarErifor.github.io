Введение


Прежде всего, необходимо узнать, как добавить сценарий JavaScript на страницу HTML. Это можно сделать одним из двух способов: поместить теги "script" на Web-странице и расположить код JavaScript внутри этих тегов, или поместить весь код JavaScript в отдельный файл и связаться с ним с помощью тега "script".

Для небольшого кода, который будет использоваться на одной странице, подойдёт и размещение его между тегами Script. В дальнейшем будет показан и второй вариант добавления сценария.

Для начала работы необходимо иметь установленный редактор кода, например Sublime Text 3 или Visual Studio Code. Запустите программу, напишите основную струкуру HTML документа и сохраните в формате HTML:

Создайте новый HTML файл с начальной струкутрой командой File -> New File.

Заполните структуру документа как в примере ниже:

<!DOCTYPE html>
<html lang="ru">
<head>
	<meta charset="UTF-8">
	<title>Заголовок</title>
	<script >
            /*Место для машего кода*/
	</script>
</head>

</body>
</html>

Сохраните файл в своей личной папке с расширением HTML командой Save, либо воспользуйтесь комбинацией клавиш "ctrl + s"

Скрипты нужно в дальнейшем писать внутри тега "script".
