Создайте страницу, которая спрашивает у пользователя имя, которую потом выводит. Создайте новый HTML файл с начальной струкутрой командой File -> New File.

Используйте функцию "Prompt" для вывода сообщения с полем ввода:

<script type="text/javascript">
   let name = prompt("Как вас зовут?");
   alert('Привет, '+ name);
</script>

Теперь попробуйте создать создать страницу с запросом имени пользователя и возраста. Затем выведите полученные результаты

Для переноса строки в функции "alert" используйте "\n\" заключив его в кавычки и добавив знаки "+" перед ним и после него.

Сохраните файл в личной папке.
