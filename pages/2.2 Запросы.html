2.2 Запросы


Для начала работы, скопируйте прошлую работу. Создайте страницу, которая спрашивает у пользователя имя, которую потом выводит.

Чтобы решить данную задачу, используйте функцию "Prompt" для вывода сообщения с полем ввода. Данная функция позволит уже задавать такую переменную, которую захочет пользователь:

1
2
3
4
<script type="text/javascript">
   let name = prompt("Как вас зовут?");
   alert('Привет, '+ name);
</script>

Задача: попробуйте создать страницу с запросом имени пользователя и возраста. Затем выведите полученные результаты.

Для переноса строки в функции "alert();", используйте "\n", заключив его в кавычки и добавив знаки "+" перед ним и после него. Пример:

alert("Вывод №1: "+ name + "\n " + "Вывод №2: " + "...");

Сохраните файл в личной папке под названием 2.2 Запросы.
