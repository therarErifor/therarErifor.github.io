Описание языка JavaScript

JavaScript – является легким, интерпретируемым, объектно-ориентированным языком с первоклассными функциями и наиболее известен как язык сценариев для веб-страниц.

Программы на этом языке называются скриптами. Они могут встраиваться в HTML и выполняться автоматически при загрузке веб-страницы.

Электронный практикум состоит из следующих разделов:

- Практикум. Содержит учебные задания, которые необходимо выполнить и требования к выполнению работ

- Контроль знаний. Содержит тест по пройденному материалу.

- Вспомогательные материалы. Содержит дополнительные материалы для изучения
