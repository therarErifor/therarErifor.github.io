6.1 Операторы циклов «While»


Для многократного повторения одного участка кода предусмотрены циклы. Чтобы понять как работает цикл «While» в JavaScript, рассмотрим примеры:

Cинтаксис цикла while:

while(/* условие. Выполняется пока верно. */){
	/* Тело цикла */
}

Пример реализации условия с оператором "While". Цикл ниже выводит i, пока i < 3:

let i = 0;
while (i < 3) { // выводит 0, затем 1, затем 2
  alert( i );
  i++;
}
