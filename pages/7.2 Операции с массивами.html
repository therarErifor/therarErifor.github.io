7.2 Операции с массивами


Давайте произведём 5 операций с массивом.

1. Создайте массив styles с элементами «Яблоко» и «Апельсин».

let styles = ["Яблоко", "Апельсин"];

2. Добавьте «Груша» в конец.

styles.push("Груша");

3. Замените значение в середине на «Слива». Ваш код для поиска значения в середине должен работать для массивов с любой длиной.

styles[Math.floor((styles.length - 1) / 2)] = "Слива";

4. Удалите первый элемент массива и покажите его.

alert( styles.shift() );

5. Вставьте «Персик» и «Гранат» в начало массива.

styles.unshift("Персик", "Гранат");

Массив по ходу выполнения операций:

Яблоко, Апельсин
Яблоко, Апельсин, Груша
Яблоко, Слива, Груша
Слива, Груша
Персик, Гранат, Слива, Груша

Сохраните файл в вашей личной папке под названием 7.2 Операции с массивами.

Примечание! Чтобы отслеживать все изменения в ходе операций с массивом, вводите команду "alert(styles);" после каждой операции в поле ввода
