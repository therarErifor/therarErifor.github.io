Контроль знаний


Уважаемый студент! Для проверки качества пройденного материала вы можете пройти итогоый тест. Начать его вы можете нажав по кнопке:

Инструкция

1. При нажатии кнопки "Начать тест" появится отдельное окно;

2. Для начала прохождения теста нажмите "Приступить к тесту";

3. По прохождению теста выбирайте один правильный ответ на каждый вопрос;

4. После прохождения теста, вам будут предоставлены количество правильных ответов, оценка, а так-же возможность начать тест заново;

Чтобы выйти из теста, нажмите кнопку закрытия окна.
