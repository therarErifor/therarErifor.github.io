Функции. Базовые операции над числами


Создайте страницу, с полями ввода для двух чисел и кнопками для произведения операции сложения, вычитания, умножения и деления.

Операции будут производиться функциями. Функции будут расположены в отдельном файле. чтобы с ним связаться, необходимо указать путь к файлу со скриптом:

<script src="script.js"></script>

Далее разместите на веб-странице следующие элементы:

Присвойте полям ввода стилевой идентификатор "id='n1'" и "id='n2'" соответсвенно

каждой кнопке присвойте событие, вызывающее фнукцию onclick="_Название функции_()" (Скобки должны быть обязательно, без них функция не будет работать). Каждая функция уже будет выполняться в файле скрипта. Пример:

<button onclick="plus()">Сложение</button>

Затем, приcвойте полю ввода результата стилевой идентификатор "id='out'" и подсказку "placeholder='Результат'" при желании:

<input id="out" placeholder="Результат">

Сохраните файл в личной папке.

Перейдём к созданию файла скрипта. Создайте новый файл "script" с расширением "js". Рекомендуется поместить документ со скриптом в отдельной папке, для удобства

Функции заполняёте по следующему примеру: 

function plus(){
	let a, b, c;
	a = document.getElementById('n1').value;
	a = parseInt(a);

	b = document.getElementById('n2').value;
	b = parseInt(b);
	c = +a + +b;
	document.getElementById('out').value = c;
}
Внимание! Цветовая схема может не соответсвовать вашему; Для сложения чисел необходимо применить унарный плюс

Метод "document.getElementById" возвращает объект Element, представляющий из себя элемент, глобальный атрибут id которого соответствует указанной строке

"parseInt" проверяет чтобы введенное значение было целочисленного типа

"value" указывает введенное значение
